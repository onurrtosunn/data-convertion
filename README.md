# data-convertion

Firstly run this: 
```
pip install -r requirements.txt
```

Prerequest:

```
       os
       csv
       glob
       json
       lxml
       numpy
       pandas
       pillow
       argparse
```
## Usage 
python3 txt_to_csv.py -i input_file_file -o output_file
* input_file (i.e. input.txt)
* output_file (i.e. output.csv)
